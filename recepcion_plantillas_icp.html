Flujo de recepción

Secuencia esperada antes de iniciar validaciones estructurales.

1
Recepción desde Outlook El bot identifica correos con adjuntos ICP del período activo.
2
Descarga del archivo Se descargan adjuntos Excel y se descartan formatos no permitidos.
3
Clasificación inicial Se clasifica por año, mes, compañía, tipo de archivo y versión.
4
Repositorio SharePoint Se almacena evidencia en carpeta estructurada para trazabilidad.
5
Envío a validación El archivo queda disponible para control de hojas, columnas, fechas y datos mínimos.

Reglas de recepción

Controles mínimos aplicados antes de procesar.

✓
Formato permitido Solo plantillas Excel oficiales: .xlsx o .xls.
✓
Clasificación obligatoria Período, compañía, tipo de envío, origen y versión son obligatorios.
✓
Reenvíos o correcciones Cuando una compañía envía una versión corregida, el sistema debe conservar trazabilidad contra el archivo recibido originalmente.
